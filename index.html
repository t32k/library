Setup

Welcome to the currently in-development MVCSS Library. This codebase is a modular framework for markup patterns (modules/) and JavaScript components (components/) that are built on top of MVCSS.

File Structure

assets/ is made up of stylesheets/ and javascripts/ (You will also include your fonts/ and images/ here).

Configuration

The assets/stylesheets/_config.sass file is where all of the configuration options are. You can use this file to tweak base variables, as well as all variables set on the modules/.

Stylesheets

core/ contains the core of MVCSS, with a few additions. modules/ is where all of the components that can be plugged into MVCSS live. As always, application.sass is your main stylesheet.

JavaScripts

javascripts/ contains an application.coffee (and compiled application.js) file for base functionality, and then components/ contains more specific JavaScript components. You can include the ones that you need, just like the Sass modules/.

Containers

Requires modules/_containers.sass

Rows

.row

Rows are full-width blocks of content. Often times, we'll create modifiers (.row--a) that have different background colors.

.row
  %h1 This is full-width content.

Cells

.cell

Cells are the inner wrappers for you site content. The modifiers (.cell--a) will setup alternate widths for your site content.

.cell
  %h1 This is content within a inner wrapper.

Cards

.card

Cards are stylistic wrappers for content elements.

.card
  %h1 This is a stylistic card.

Buckets

.bucket--01

Buckets are content elements with a media element floated to the left, and content floated next to that media element. There are modifiers for the buckets based on the $bucket-sizes variable, which allows you to pass in the common media element sizes. For instance, .bucket--01 accounts for a 75px-wide media element.

.bucket--01
  %img.bucket-media(src="http://placehold.it/75x75" alt="placeholder")
  .bucket-content
    %h1 Bucket
    %p This is a bucket.

Basic Form

.form

I hope you remember your name, because you need to enter it.

Forgot your password?

%form.form(methd="post" action="")
  %fieldset.form-field
    %label.form-label(for="name") Name: *
    %input.form-input(type="text" id="name" placeholder="Enter your name")
    %p.form-note I hope you remember your name, because you need to enter it.

  %fieldset.form-field
    %label.form-label(for="email") Email: *
    %input.form-input(type="email" id="email" placeholder="Enter your email")

  %fieldset.form-field
    %label.form-label(for="password") Password: *
    %input.form-input(type="password" id="password")
    %p.form-note
      %a(href="#") Forgot your password?

  %fieldset.form-field
    %label.form-label(for="select") Select your lunch today:
    %select.form-select
      %option(value="empty") Choose one:
      %option(value="item") Hamburger and fries
      %option(value="item") Salad
      %option(value="item") Enchilada
      %option(value="item") Salmon

  %fieldset.form-field
    %label.form-label(for="radio") Select your gender:
    %ul
      %li
        %label.form-label.form-label--radio(for="male")
          %input(type="radio" name="gender" id="male") Male
      %li
        %label.form-label.form-label--radio(for="female")
          %input(type="radio" name="gender" id="female") Female
      %li
        %label.form-label.form-label--radio(for="other")
          %input(type="radio" name="gender" id="other") Other

  %fieldset.form-field
    %label.form-label(for="notes") Extra notes:
    %textarea.form-input(id="notes")

  %fieldset.form-field
    %label.form-label(for="select") Select an option:
    %label.form-label.form-label--check(for="check")
      %input(type="checkbox" id="check") Yes, I would like to receive tri-daily emails from your service.

  %fieldset.form-field.form-field--submit
    %input.btn--a(type="submit" value="Submit")

Silent Classes

If you don't want to explicitly call the .grid-box-- modifier classes in your HTML, you can set the $grid-silent variable to true to use Sass placeholder selectors, like so:

.main
  @extend %grid-box--2of3
.sidebar
  @extend %grid-box--1of3

Responsive Grid

In order to setup breakpoints for the grid, you'll need to set the $grid-defaults variable for your specified breakpoints. Then, either in your HTML or through @extends (when using $grid-silent), you'll do something like this:

.grid
  .grid-box.grid-box--desk--2of3.grid-box--lap--1of1
    %h1 Main
  .grid-box.grid-box--desk--1of3.grid-box--lap--1of1
    %h2 Sidebar
